# Kom igång som utvecklare

## Hämta hem en komponent

Installera komponenter via `npm install @migrationsverket/[component]`. Komponentbiblioteket ligger på Nexus, se till
att du hämtar via rätt registry om det inte redan är inställt.

``` bash
// .npmrc
@migrationsverket:registry=http://nexus.migrationsverket.se/repository/npm/
```

## Importera och använd komponenten

Generellt, importera och använd direkt:

```tsx
import { Component } from '@migrationsverket/component';

<Component/>
```

För specfika instruktioner, se respektive [komponent](../components/)

## Installera globala styles

För att få tillgång till globala styles som t.ex typsnittet Inter 
    1. `npm install @migrationsverket/theme`
    2. `import '@migrationsverket/theme/index.css` globalt i din app
