# Att utveckla med designsystemet

## Hämta hem komponent

Installera komponenter via `npm install @midas-ds/[component]`. Komponentbiblioteket läses in av verkets vanliga registry, se till
att du hämtar via rätt registry om det inte redan är inställt.

## Importera och använd komponent

Generellt, importera och använd direkt:

```tsx
import { Component } from '@midas-ds/component';
...
<Component/>
```

För specfika instruktioner, se respektive [komponent](../../components/)

## Installera globala styles

För att komponenterna ska renderas exakt korrekt behövs lite global css. Detta finns i paketet `@midas-ds/theme`.

```bash npm2yarn
npm install @midas-ds/theme
```
```tsx {1} title="App.tsx (rootfilen i din app)"
import "@midas-ds/theme/index.css";

export default function App({children}) {
  return <main>
    {children}
  </main>
}

export default App;
```
