# Att utveckla med designsystemet

## Hämta hem komponent

Installera komponenter via `npm install @migrationsverket/[component]`. Komponentbiblioteket läses in av verkets vanliga registry, se till
att du hämtar via rätt registry om det inte redan är inställt.

## Importera och använd komponent

Generellt, importera och använd direkt:

```tsx
import { Component } from '@migrationsverket/component';

<Component/>
```

För specfika instruktioner, se respektive [komponent](../../components/)

## Installera globala styles

För att få tillgång till globala styles som t.ex typsnittet Inter 
    1. `npm install @migrationsverket/theme`
    2. `import '@migrationsverket/theme/index.css` globalt i din app

## Är det något knas?

Saknar du komponenter eller features i nuvarande komponentbibliotek? Har du hittat en bugg? Ta kontakt med Designsystemet på Mattermost så hjälper vi dig att skapa ett ärende!
