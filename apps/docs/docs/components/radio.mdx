# Introduktion

Radioknappar är en typ av input-fält som används för att välja **ett** alternativ från en uppsättning av flera alternativ. Om det ska gå att välja flera eller inget alternativ är det en [Kryssruta](/components/checkbox) som ska användas
Om det är fler alternativ sju (7) används med fördel en [Rullgardinsmeny](/components/select) istället.

<RadioGroup label={'Är det ok med ananas på pizza?'} description={'Frukt på pizza är en vattendelare'} defaultValue={1}>
  <Radio value={1}>Ja, Hawaii är ju jättegott</Radio>
  <Radio value={2}>Nej! Det dör en italienare varje gång någon har ananas på pizza!</Radio>
</RadioGroup>

## Installation och användning

```bash npm2yarn
npm install @migrationsverket/radio
```

```tsx
import { Radio, RadioGroup } from '@migrationsverket/radio';
```

### Normal användning

Gruppera flera `<Radio>` i en `<RadioGroup>` och sätt rubrik och beskrivande text.

---

<RadioGroup label={'Skoval'} description={'Välj lämplig fotbeklädnad'} defaultValue={2}>
  <Radio value={1}>Sneakers</Radio>
  <Radio value={2}>Sandaler</Radio>
  <Radio value={3}>Kängor</Radio>
</RadioGroup>

```tsx
<RadioGroup label={'Skoval'} description={'Välj lämplig fotbeklädnad'} defaultValue={2}>
  <Radio value={1}>Sneakers</Radio>
  <Radio value={2}>Sandaler</Radio>
  <Radio value={3}>Kängor</Radio>
</RadioGroup>
```

### Disabled

Sätt `isDisabled` antingen på `RadioGroup` eller på enskilda val av `Radio`

<RadioGroup label={'Skoval'} description={'Välj lämplig fotbeklädnad'} defaultValue={1} >
  <Radio value={1}>Sneakers</Radio>
  <Radio value={2} isDisabled>Sandaler</Radio>
  <Radio value={3}>Kängor</Radio>
</RadioGroup>

### Validering

Sätt validering med `isInvalid` eller använd inbyggda tester som `isRequired`.

## Riktlinjer

---

- En av radioknapparna ska alltid vara ifylld. Om du misstänker att användaren vill kunna avstå från att välja något, skapa ett sista alternativ som du kallar "Inget av ovanstående" eller liknande.
- Givet ovanstående riktlinje kan radioknappar inte användas ensamma, utan måste alltid vara två eller flera.
- Radioknapparna placeras som regel vertikalt för att underlätta avläsning. Om antalet alternativ för en given fråga är begränsat till två (2) och det är ont om vertikalt utrymme så kan en horisontell orientering användas.
- Fältetiketten ska inledas med stor bokstav och inte följas av punkt.
