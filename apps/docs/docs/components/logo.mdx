# Introduktion

På webbsida och e-tjänster ska Migrationsverkets officiella logotyp, den röda bron tillsammans med namnet, finnas.

## Installation och användning

```bash npm2yarn
npm install @migrationsverket/logo
```

```tsx
import { Logo } from '@migrationsverket/logo';
```

I de situationer där det inte funkar med React så kan du [ladda ner logotypen som SVG här (komprimerad fil)](/img/logo/Mig_logo_SVG.zip)
