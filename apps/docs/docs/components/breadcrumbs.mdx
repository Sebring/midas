## Introduktion

Brödsmulor används för att visa användaren vart hen är i ett djupt navigationsträd.


## Installation och användning

```bash npm2yarn
npm install @midas-ds/breadcrumbs
```

```tsx
import { Breadcrumbs } from '@midas-ds/breadcrumbs';
```

```tsx live
function App() {
  const items = [
      {
        "title": "Jag vill förlänga",
        "href": "#"
      },
      {
        "title": "Arbete",
        "href": "#"
      },
      {
        "title": "Anställd",
        "href": "#"
      }
  ];

  return (
    <Breadcrumbs items={items} />
  )
}
```
