# Introduktion

Använd ett inmatningsfält när användaren ska fylla i kortare information, tex namn, personnummer eller epostadress. För längre input används TextArea (Coming soon).

Komponenten bygger på [TextField](https://react-spectrum.adobe.com/react-aria/TextField.html) från [React Aria](https://react-spectrum.adobe.com/react-aria/index.html)


<TextField name={'nameExample'} label={'Fyll i ditt namn'} description={'För och efternamn'} isRequired/>

## Installation och användning

```bash npm2yarn
npm install @migrationsverket/textfield
```

```tsx
import { TextField } from '@migrationsverket/textfield';
```

---

### Etikett och beskrivning

Använd `label` för etikett och `description` för valfri extra beskrivning.

<TextField label={'Etikett'} description={'Beskrivning'} name={'example'}/>

```tsx
<TextField label={'Etikett'} description={'Beskrivning'} />
```

### Validering

TextField validerar automatiskt på `type` och `isRequired`

<TextField
  name={'email'}
  label={'Skriv in din e-post'}
  description={'Obligatorisk!'}
  type={'email'}
/>

```tsx wrap=true
<TextField
  label={'Skriv in din e-post'}
  description={'Obligatorisk!'}
  isRequired
  type={'email'}
/>
```

Det går också att definiera en egen funktion för validering:

<TextField
  name={'icecream'}
  label={'Skriv in "glass"'}
  validate={(value) => (value === 'glass' ? true : 'Du måste skriva glass!')}
/>

```tsx
<TextField
  label={'Skriv in glass'}
  validate={(value) => (value === 'glass' ? true : 'Du måste skriva glass!')}
/>
```

### Lösenord

Sätt `type={'password'}` för att kunna visa och dölja inmatad text.

<TextField type={'password'} name={'pw'} label={'Skriv in ditt lösenord'} autoComplete={'off'}/>
