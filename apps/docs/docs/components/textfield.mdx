## Introduktion

Använd ett inmatningsfält när användaren ska fylla i kortare information, tex namn, personnummer eller epostadress. För längre input används TextArea.

## Installation och användning

```bash npm2yarn
npm install @midas-ds/textfield
```

```tsx
import { TextField } from '@midas-ds/textfield';
```

```tsx live
<TextField label={'Etikett'} description={'Beskrivning'} />
```

### Validering

TextField validerar automatiskt på `type` och `isRequired`

```tsx live wrap=true
<TextField
  label={'Skriv in din e-post'}
  description={'Obligatorisk!'}
  isRequired
  type={'email'}
/>
```

Det går också att definiera en egen funktion för validering:

```tsx live
<TextField
  label={'Skriv in glass'}
  validate={(value) => (value === 'glass' ? true : 'Du måste skriva glass!')}
/>
```

### Lösenord

Sätt `type={'password'}` för att kunna visa och dölja inmatad text.

```tsx live
<TextField type={'password'} name={'pw'} label={'Skriv in ditt lösenord'} autoComplete={'off'}/>
```

### Desktop
Anpassa längden på inputfältet, tex med [Grid](./grid.mdx) eller [Flex](./flex), så att det matchar förväntad längd på det användaren ska mata in. Ett svenskt postnummer är alltid 5 siffror och då bör fältet vara kort medan ett fält där användaren ska fylla i alla sina namn behöver vara långt

```tsx live
<Grid>
    <div style={{ gridColumn: 'span 2' }}>
          <TextField name={'nameExample'} label={'Postnummer'} /> 
    </div>
    <div style={{ gridColumn: 'span 6' }}>
          <TextField name={'nameExample'} label={'Fyll i alla dina namn'} /> 
    </div>
</Grid>
```
