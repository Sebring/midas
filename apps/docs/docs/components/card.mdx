# Introduktion

Kortkomponenten är en yta med plats för information och åtgärder relaterat till ett specifikt ämne. Kort används primärt för att presentera innehåll på ett visuellt tilltalande sätt. 

Ange `background` för en blå bakgrund. Använd `image={source='url', description={'beskrivning}}` för att visa en bild överst på komponenten.
Hela ytan är klickbar och interaktioner hanteras med `onPress()`.
