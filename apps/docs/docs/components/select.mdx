# Introduktion

En väljare visar en hopfällbar lista med alternativ och låter en användare välja ett av dem.

<Select label={'Frukt'} description={'Välj en frukt'}><Item>Apelsin</Item><Item>Banan</Item><Item>Mango</Item></Select>
