## Introduktion

En väljare visar en hopfällbar lista med alternativ och låter en användare välja ett av dem.


## Installation och användning

```bash npm2yarn
npm install @migrationsverket/select
```

```tsx
import { Select } from '@migrationsverket/select'
```

```tsx live
function App() {
    const options = [
        { id: 'apple', name: 'Apple' },
        { id: 2, name: 'Banana' },
        { id: 3, name: 'Kiwi' },
        { id: 4, name: 'Elderflower' },
        { id: 5, name: 'Nectarine' },
        { id: 6, name: 'Watermelon' },
        { id: 7, name: 'Pineapple' },
        { id: 8, name: 'Apricot' },
        { id: 9, name: 'Cloudberry' },
    ];

    return (
        <Select
            label="Välj din favorit"
            defaultSelected="Välj frukt"
            items={options}
        >
            {(item) => <Item>{item.name}</Item>}
        </Select>
    )
}
```
