# Introduktion
Filuppladdning används när användaren behöver ladda upp en eller flera filer från det egen filsystemet.

:::info
Filuppladdning är i experimentell status för UX-testning och fungerar inte rent tekniskt. Det är rimligt att anta att både implementation och uppbyggnad av template kommer att förändras.
Komponenten bygger på [React Aria FileTrigger](https://react-spectrum.adobe.com/react-aria/FileTrigger.html)
