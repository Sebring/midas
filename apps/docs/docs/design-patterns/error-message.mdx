# Felmeddelande

En sammanfattning av de fel som förekommer visas högst upp på sidan. Sammanfattningen ska bestå av förklaring
på vad som är fel och konkreta råd på hur det/de går att åtgärda för att undvika nya fel.

Sammanfattningen går endast att ta bort genom att åtgärda de fel som anges. Det ska inte finnas någon kryss-knapp
för stängning.

Felmeddelande
När det sker ett fel eller om det saknas obligatorisk information ett felmeddelande. Meddelandet ska förklara
vad som är fel och hur det går att åtgärda.

Gör såhär:

- Felmeddelandet skrivs med text i rött och placeras nedanför den plats där felet uppstod. Meddelandet inleds
  med ikonen för fel i rött.
- En röd vertikal linje i vänstermarginalen markerar det område som innehåller felet, inklusive frågan eller etiketten, felmeddelandet och fältet.
- Om felet berör ett specifikt inmatningsfält ska även detta markeras med en röd ram.

Felmeddelande i tabeller:

Felmeddelandet ska placeras under inmatningsfältet.
