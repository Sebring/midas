# Fokus

Fokusmarkering används för att visa användaren vilken komponent som har tabbfokus. Detta är till hjälp när användare navigerar på sidan med tangentbord eller andra tillgänglighetsverktyg. Fokus ska finnas tillgängligt på alla komponenter.

Fokus består av två ramar som läggs runt komponenten, en 2px vit innerram och en 2px Svart (#000000) ytterram på stora komponter. På små komponenter så är det 1px innerram och 2px ytterram
